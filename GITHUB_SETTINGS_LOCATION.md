# GitHub 조직 설정 위치 가이드

## 현재 위치
NeuralStudioKr 조직 Overview 페이지

## 설정 페이지 찾는 방법

### 방법 1: 직접 URL로 접근 (가장 빠름)

#### Railway 앱 설치 페이지:
```
https://github.com/organizations/NeuralStudioKr/settings/installations
```

#### 조직 설정 메인 페이지:
```
https://github.com/organizations/NeuralStudioKr/settings/profile
```

### 방법 2: 탭 메뉴에서 찾기

1. **현재 보이는 탭들**:
   - Overview (현재 선택됨)
   - Repositories
   - Projects
   - Packages
   - Teams
   - People

2. **Settings 탭 찾기**:
   - 탭 목록 끝에 "Settings" 탭이 있을 수 있음
   - **주의**: Owner 또는 Admin 권한이 있어야 보입니다
   - 권한이 없으면 탭이 보이지 않을 수 있음

### 방법 3: 상단 메뉴에서 찾기

1. **조직 이름 옆 아이콘**:
   - 조직 이름 "NeuralStudioKr" 옆에 ⚙️ (설정) 아이콘 클릭
   - 또는 조직 프로필 사진 옆 메뉴

2. **프로필 드롭다운**:
   - 오른쪽 상단 프로필 아이콘 클릭
   - "Settings" 또는 "Organization settings" 찾기

### 방법 4: 검색 기능 사용

1. 상단 검색창에 `/` 입력
2. "Settings" 또는 "Organization settings" 검색
3. 결과에서 조직 설정 선택

## Railway 앱 설치 페이지 (직접 링크)

**가장 빠른 방법**:
```
https://github.com/organizations/NeuralStudioKr/settings/installations
```

이 페이지에서:
1. Railway App 찾기
2. "Configure" 또는 "Grant access" 클릭
3. 조직 저장소 접근 권한 승인

## 권한이 없는 경우

만약 "Settings" 탭이 보이지 않는다면:

1. **권한 확인**:
   ```
   https://github.com/orgs/NeuralStudioKr/people
   ```
   - 자신의 역할이 "Owner" 또는 "Admin"인지 확인

2. **조직 관리자에게 요청**:
   - "People" 탭에서 Owner 찾기
   - Owner에게 Railway 앱 설치 요청

## 빠른 링크 모음

- **조직 설정 메인**: https://github.com/organizations/NeuralStudioKr/settings/profile
- **앱 설치 페이지**: https://github.com/organizations/NeuralStudioKr/settings/installations
- **Third-party access**: https://github.com/organizations/NeuralStudioKr/settings/installations
- **조직 멤버**: https://github.com/orgs/NeuralStudioKr/people

## 다음 단계

설정 페이지에 접근한 후:

1. **Third-party access** 또는 **Installed GitHub Apps** 섹션 찾기
2. **Railway App** 검색
3. **Configure** 또는 **Grant access** 클릭
4. 저장소 접근 권한 설정 (`insta_selftest` 선택)
5. **Save** 또는 **Install** 클릭
